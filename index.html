About Monero Nodes & This Directory

Running your own full node is always the most secure option. However, if you need immediate wallet access or are on a mobile device, using reliable Monero public nodes is the next best choice. This page tracks a community-driven public Monero nodes feed, helping users easily connect to high-uptime Monero remote nodes while filtering out endpoints for better privacy hygiene.

Frequently Asked Questions

What are Monero nodes (XMR nodes)?
A Monero node is a computer running the Monero software to validate and broadcast transactions. Using a Monero remote node lets a lightweight wallet query blockchain data to safely send and receive XMR without downloading the entire blockchain locally.

How do I use a public Monero node?
Simply copy the Node URL from the list above and paste it into the "Remote Node" settings of your Monero wallet (like Cake Wallet, Monerujo, or the official GUI wallet).
